# 🏨 Hotel Performance Analysis Dashboard

A comprehensive end-to-end Business Intelligence project built using **Power BI** to analyze hotel booking dynamics, revenue drivers, cancellation behaviors, and operational metrics.

---

## 📊 Project Overview
This dashboard provides deep insights into hotel performance data, helping stakeholders track key metrics, identify cancellation trends, and understand regional revenue distribution.

### Key Pages & Features:
1. **Hotel Performance Analysis:** High-level KPIs (Total Revenue of $42.72M, 119K Total Bookings), geographic revenue distribution map, and country-level breakdowns.
2. **Booking Dynamics:** Market segment mix (Online TA, Offline TA, Groups), monthly booking trends, guest segments, and booking status proportions.
3. **Operational Overview:** Cancellation behavior based on lead time, comparison between City Hotels and Resort Hotels, and timeline revenue trends.
4. **Root Cause Analysis:** Utilizing a Decomposition Tree to break down booking drivers by hotel type, country, and market segments.

---

## 🛠️ Tools & Technologies Used
* **Power BI:** Data visualization, DAX measures, Power Query.
* **Data Modeling:** Star schema structure and relationship building.
